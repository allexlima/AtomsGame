 Atom's Game is developed with Javascript and HTML5 which aims to help children and teenagers in the learning of elements of the periodic table. This project was presented in National Institute of Amazon Researches (INPA) for teachers and students from public schools in Manaus, AM - Brazil. The project team, aims expand the game in order to contribute with the chemistry learning through digital and attractive games to child audience.
